# LMK Tester Docs

此仓库包含 LMK Tester 项目的文档网站。

## 项目简介

LMK Tester 是一款 Android 应用恢复测试工具，通过模拟内存压力触发系统 LMK（Low Memory Killer），帮助开发者验证应用在被系统回收后的恢复能力。
